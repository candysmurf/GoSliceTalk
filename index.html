Length vs Capacity

Slice grows and shrinks

  • Slice is accessible from index 0 to len(s)-1
  • Slice s[x:y] is sliceable if x <= y <= cap(s)
  • Slice is always appendable

Why capacity

doubles

  • Insertion is O(n) when capacity doubles
  • Insertion is O(n^2) when capacity increases linearly


The End

Go Slice

  • len(s) <= cap(s)
  • access slice from index 0 to len(s)-1
  • slice s[x:y] if x <= y <= cap(s)
  • append doubles capacity when the capacity is exceeded
  • define length and capacity when you know the data size
